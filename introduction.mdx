## Prerequisites

Install the following languages: goland and node js.

Then install:

1. **Ganache** - Ganache is used for running a local blokchain network.
2. **solc, abigen** -  solc and abigen are used for deploying smart contracts to hardhat network and interacting with the deployed smart contract.

## <Tooltip tip="">Installing </Tooltip>Ganache

Run the following commands to install ganache locally:

<CodeGroup>

```bash npm
npm install --global ganache
```

</CodeGroup>

## Installing solc and abigen

Follow instructions given in https://goethereumbook.org/en/smart-contract-compile/. The abigen tool can be installed using the following command:

<CodeGroup>

```bash golang
go install github.com/ethereum/go-ethereum/cmd/abigen@latest
```

</CodeGroup>

## Run blockchain network locally

<CodeGroup>

```bash npm
ganache
```

</CodeGroup>

## Run the golang program

<CodeGroup>

```bash golang
go build untrace
```

</CodeGroup>
