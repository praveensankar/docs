## Prerequisites

Install the following languages: goland and node js.

Then install:

1. **Harhat** - Hardhat is used for running a local blokchain network.
2. **solc, abigen** -  solc and abigen are used for deploying smart contracts to hardhat network and interacting with the deployed smart contract.

## <Tooltip tip="">Installing hardhat</Tooltip>

Run the following commands to install hardhat locally:

> `npm install --save-dev hardhat`
>
> `npx hardhat init`

Select Create an empty hardhat.config.js with your keyboard and hit enter.

Install the following plugins:

> `npm install --save-dev @nomicfoundation/hardhat-toolbox`\
> `npm install --save-dev @nomicfoundation/hardhat-ignition-ethers`

Refer to the official hardhat tutorial if you encounter any errors in the installation of hardhat:

https://hardhat.org/tutorial/creating-a-new-hardhat-project

## Installing solc and abigen

Follow instructions given in https://goethereumbook.org/en/smart-contract-compile/. The abigen tool can be installed using the following command:

> `go install github.com/ethereum/go-ethereum/cmd/abigen@latest`

## Run blockchain network locally

> `npx hardhat node`

## Run the golang program

> `go build untrace`
