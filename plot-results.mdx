## Follow the instructions below to plot results

Change the current directory to plots

<CodeGroup>

```bash terminal
cd plots
```

</CodeGroup>

Install dependencies (only for the first time)

<CodeGroup>

```bash terminal
poetry install
```

</CodeGroup>

Then run the main.py file

<CodeGroup>

```bash terminal
poetry run python main.py
```

</CodeGroup>
